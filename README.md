# ShopHub One - Lagos Store

Single-deployment SvelteKit e-commerce for Nigeria.

## Deploy to Render (1-click)

1. Push to GitHub
2. Render → New Web Service → Connect repo
3. Build: `npm install && npm run build`
4. Start: `node build`
5. Deploy
